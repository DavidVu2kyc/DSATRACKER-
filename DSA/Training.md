1-Week Java DSA Training

  Day 1
  Arrays, strings, two pointers, sliding window.
  Practice:

  1. Two Sum
  2. Valid Palindrome
  3. Best Time to Buy and Sell Stock
  4. Longest Substring Without Repeating Characters

  Java focus:

  - int[]
  - String
  - StringBuilder
  - HashMap

  Day 2
  Hashing and sorting.
  Practice:

  1. Contains Duplicate
  2. Valid Anagram
  3. Group Anagrams
  4. Top K Frequent Elements

  Java focus:

  - HashMap
  - HashSet
  - Arrays.sort()
  - custom comparator

  Day 3
  Linked list, stack, queue.
  Practice:

  1. Reverse Linked List
  2. Middle of Linked List
  3. Valid Parentheses
  4. Implement Queue using Stacks

  Java focus:

  - custom ListNode
  - Deque
  - ArrayDeque

  Day 4
  Binary search, recursion, backtracking.
  Practice:

  1. Binary Search
  2. Search Insert Position
  3. Subsets
  4. Permutations

  Java focus:

  - recursion
  - base case
  - helper functions

  Day 5
  Trees and traversal.
  Practice:

  1. Maximum Depth of Binary Tree
  2. Same Tree
  3. Invert Binary Tree
  4. Binary Tree Level Order Traversal

  Java focus:

  - custom TreeNode
  - DFS
  - BFS with queue

  Day 6
  Heap, greedy, graph basics.
  Practice:

  1. Kth Largest Element in an Array
  2. Last Stone Weight
  3. Number of Islands
  4. Flood Fill

  Java focus:

  - PriorityQueue
  - BFS/DFS
  - visited tracking

  Day 7
  Revision and mock test.

  - Spend 2 to 3 hours daily.
  - First solve brute force, then optimize.
  - After each problem, write:
      - approach
      - time complexity
      - space complexity
  - If stuck more than 20 to 25 minutes, read hints, then retry.
  - Recode important patterns from memory:
      - binary search
      - BFS
      - DFS
      - linked list reverse
      - sliding window

  Java Templates You Should Know

  - HashMap frequency count
  - HashSet lookup
  - ArrayDeque for stack/queue
  - PriorityQueue min/max heap
  - DFS recursion template
  - BFS queue template

  Best Habit
  Keep one notebook or file with:

  - problem name
  - pattern used
  - mistake made
  - better solution idea
