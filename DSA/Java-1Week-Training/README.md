# Java 1-Week DSA Training

This folder is a simple structure for practicing DSA in Java for 7 days.

## Structure

- `day01-arrays-strings` to `day07-mock-revision`: daily practice folders
- `templates`: reusable Java templates
- `notes`: progress tracking and mistakes log

## How To Use

1. Read the `tasks.md` file for the day.
2. Solve the problems in the `src` folder.
3. Write notes in `notes/progress-log.md`.
4. Reuse code from `templates` when needed.

## Suggested Daily Routine

- 20 min: review concept
- 60 min: solve 2 easy problems
- 45 min: solve 1 medium problem
- 20 min: review mistakes and update notes

## Compile Example

```bash
javac src/*.java
java -cp src Problem01_TwoSum
```
