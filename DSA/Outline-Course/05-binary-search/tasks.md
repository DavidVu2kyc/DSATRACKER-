# Topic: Binary Search

## Exercises

1. Binary Search
   - Search for a target in a sorted array.
   - Hint: keep the search space inclusive and update `left` and `right` carefully.

2. Search Insert Position
   - Return the index where the target exists or should be inserted.
   - Hint: binary search still works even when the target is missing.

3. First Bad Version
   - Find the first bad version with the fewest checks.
   - Hint: when `mid` is bad, keep it in the search space.

4. Find Peak Element
   - Return any peak element index.
   - Hint: compare `mid` with `mid + 1` to decide which half must contain a peak.
