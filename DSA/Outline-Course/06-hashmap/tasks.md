# Topic: HashMap

## Exercises

1. Contains Duplicate
   - Check whether any value appears at least twice.
   - Hint: a set is enough for this one.

2. Valid Anagram
   - Determine whether two strings are anagrams.
   - Hint: count character frequencies and compare.

3. Top K Frequent Elements
   - Return the `k` most frequent elements.
   - Hint: first build a frequency map, then think about sorting or heaps.

4. Longest Substring Without Repeating Characters
   - Find the length of the longest substring with unique characters.
   - Hint: combine a hash map with a sliding window.
