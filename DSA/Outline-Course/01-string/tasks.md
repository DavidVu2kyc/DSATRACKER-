# Topic: String

## Exercises

1. Reverse String
   - Given a string `s`, return the reversed string.
   - Hint: use two pointers or convert to a list and swap from both ends.

2. Valid Palindrome
   - Given a string, check whether it reads the same forward and backward after removing non-alphanumeric characters.
   - Hint: move two pointers from left and right, skipping invalid characters.

3. Longest Common Prefix
   - Given an array of strings, return the longest common prefix.
   - Hint: use the first string as a baseline and compare character by character.

4. String to Integer (`atoi`)
   - Convert a string into a 32-bit signed integer.
   - Hint: handle spaces, sign, digits, and overflow in separate steps.
