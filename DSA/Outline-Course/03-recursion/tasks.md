# Topic: Recursion

## Exercises

1. Factorial
   - Compute `n!`.
   - Hint: define the base case before writing the recursive step.

2. Fibonacci Number
   - Return the `n`th Fibonacci number.
   - Hint: write the recursive version first, then think about why it is slow.

3. Power Function
   - Compute `x^n`.
   - Hint: for negative powers, invert the result; for faster code, use divide and conquer.

4. Generate Subsets
   - Return all subsets of a list of numbers.
   - Hint: at each index, choose include or exclude.
