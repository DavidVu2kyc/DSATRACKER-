# Topic: Sorting

## Exercises

1. Sort Colors
   - Sort an array containing only `0`, `1`, and `2`.
   - Hint: try counting first, then study the three-pointer approach.

2. Merge Intervals
   - Given intervals, merge all overlapping ones.
   - Hint: sort by start time first.

3. Kth Largest Element in an Array
   - Find the kth largest value.
   - Hint: sorting solves it directly; later compare that with heap-based solutions.

4. Sort Characters by Frequency
   - Sort characters in a string by descending frequency.
   - Hint: count first, then sort by frequency.
