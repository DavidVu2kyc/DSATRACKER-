# Topic: Array

## Exercises

1. Two Sum
   - Given an array and a target, return indices of two numbers whose sum equals the target.
   - Hint: first solve with nested loops, then optimize with a hash map.

2. Remove Duplicates from Sorted Array
   - Remove duplicates in-place and return the new length.
   - Hint: keep one pointer for the next write position.

3. Best Time to Buy and Sell Stock
   - Find the maximum profit from one buy and one sell.
   - Hint: track the minimum value seen so far.

4. Move Zeroes
   - Move all zeroes to the end while keeping the order of non-zero elements.
   - Hint: use one pointer to place the next non-zero value.
